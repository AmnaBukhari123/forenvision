# ForenVision 
AI-powered Forensic Investigation & Crime Scene Reconstruction System  

## Overview
ForenVision is a web-based platform that enables investigators to manage cases, analyze forensic data, and generate structured reports.  
It combines **React.js** (frontend), **FastAPI** (backend), and **PostgreSQL** (database).  
Future integrations will include AI/ML modules for **object detection**, **blood spatter analysis**, and **2D ‚Üí 3D scene reconstruction**.
